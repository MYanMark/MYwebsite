Student-Tutor Website

This full-stack web portal was developed as part of a team project, where I served as the primary developer responsible for most of the technical implementation. The website supports student-tutor interactions with secure, role-based access and task management functionality.

Frontend Development:

I built responsive dashboards, assignment views, and profile pages using HTML, CSS, and JavaScript, focusing on interactive UI and user-friendly design.

Backend & API:

I developed RESTful APIs using Node.js (Express) to handle authentication, session control, role-based routing, and assignment/event workflows.

Database Design:

I designed and implemented a relational database using SQL Server (SSMS), including schemas for users, assignments, groups, and submissions. CRUD functionality was applied throughout.

Feature Highlights:

Key features include login/sign-up validation, role-based access control, assignment creation and management, grouping logic, and event scheduling. These components came together to create a fully usable and secure system that closely resembles a real-world application.
